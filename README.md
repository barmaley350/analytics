# Как пользоваться 

Этот репозиторий представляет собой практическое руководство, которое поможет вам перейти от основ SQL к более сложным концепциям. В нем используется пример базы данных Chinook — реалистичный набор данных, моделирующий цифровой медиамагазин (похожий на iTunes). Он включает в себя таблицы с данными об исполнителях, альбомах, треках, клиентах, счетах и т. д., что делает его идеальным для отработки запросов, объединений, агрегирования и подзапросов.
